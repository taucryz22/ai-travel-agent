# AI Travel Agent

AI Travel Agent — это веб-приложение для генерации персональных туристических маршрутов по текстовому запросу пользователя. Система анализирует естественный язык, определяет город маршрута, интересы пользователя, темп поездки и бюджет, после чего подбирает точки интереса, рассчитывает перемещения между ними и формирует готовый план поездки.

## Основные возможности

- генерация маршрута по текстовому запросу
- определение города назначения, в том числе в запросах вида «из города A в город B»
- поддержка интересов пользователя: достопримечательности, еда, искусство, прогулки, бары, культурная программа и др.
- учёт бюджета пользователя
- логика выбора способа перемещения:
  - пешком
  - общественный транспорт
  - такси
- ограничение слишком длинных пеших переходов
- фильтрация слишком далёких точек до этапа маршрутизации
- экспорт результата в `.txt`
- экспорт результата в `.pdf`
- просмотр маршрута по дням
- ссылки на Яндекс.Карты для переходов между точками

## Технологии

### Frontend
- React
- TypeScript
- Vite

### Backend
- Python
- FastAPI
- Pydantic

### Внешние и локальные сервисы
- Ollama + Qwen — для разбора пользовательского запроса
- OpenStreetMap / Nominatim — для поиска и валидации городов и мест
- OpenRouteService — для расчёта времени и расстояния между точками
- Wikivoyage — как источник текстовых подсказок для маршрута
