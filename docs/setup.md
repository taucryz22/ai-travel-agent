# Запуск проекта

## Требования

Для запуска проекта необходимы:

- Python 3.11+
- Node.js + npm
- Ollama
- модель Qwen в Ollama
- ключ OpenRouteService
- доступ в интернет для OSM / ORS / Wikivoyage
